# Sentinel Shield Advanced Intrusion Detection and Web Protection System                    ## What it does A Python-based Web Application Firewall (WAF) that detects and blocks SQL injection, XSS, path traversal, command injection, and brute-force attacks in real time.     ## Tech stack - Python 3.12, Flask, Gunicorn - Docker, Docker Compose - Nginx reverse proxy ## How to run ```bash cp .env.example .env docker build -t sentinel-shield . docker run -d --name sentinel -p 8000:8000 --env-file .env sentinel-shield ``` ## API endpoints - GET /health — health check - POST /inspect — submit request for WAF inspection - GET /dashboard — live security statistics - GET /logs — recent log entries
